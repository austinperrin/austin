# Asset Contract

Update `assets.json` after moving final sprite files into these folders. Keep the manifest keys stable so gameplay code does not need to know final file names.

- `boss`: Austin idle, move, attack, hurt, and defeated sprites.
- `cursor`: cursor normal, hurt, and defeated sprites.
- `effects`: sword slash, fireball, hit, and explosion effects.
- `items`: optional poke or pointer item art.
- `backgrounds`: optional arena background.
- `audio`: optional sound effects.

The scaffold includes SVG placeholders so the game can run before final assets are added.
